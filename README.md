# jp-text-converter
[![License](https://img.shields.io/github/license/lassjs/lass.svg)](LICENSE)

jp-text-converter是一款十分方便使用的日文转换注音工具，主要针对日文文本，进行到平假名、片假名的转换，并支持注音假名、送假名（旁注音）等注音模式。

## 特性
- 日文文本 => 平假名、片假名
- 支持注音假名和送假名
- 使用kuromoji词法分析器
- 实用日语工具
